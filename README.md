# Console-based-guessing-game
Built a console-based "Guess the Number" game using JavaScript. 
How it works: The computer picks a random integer between 1 and 100. 
The player keeps entering guesses via prompts until they find the correct number.
After each guess, the game tells the player whether their guess is too high, too low, or correct.
As a bonus, it tracks how many attempts it took to guess the number.
